## 1. Brand Overview

### 1.1 Brand Story

**Disbursify Dash** was born from a simple observation: Nigerian entrepreneurs don't run single businessesâ€”they run portfolios. Yet every financial tool treats them like they have just one.

We built Dash to be the command center for Africa's multi-business owners. A place where complexity becomes clarity, and uncertainty becomes insight.

### 1.2 Brand Essence

**One word:** Clarity

### 1.3 Brand Promise

> "See your complete financial picture in 30 seconds."

## 4. Typography

### 4.1 Font Family

**Primary Font:** Inter  
*Clean, modern, highly legible, designed for screens*

**Fallback Stack:** `Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif`

### 4.2 Type Scale

| Level | Size | Weight | Line Height | Use |
|-------|------|--------|-------------|-----|
| **Display** | 48px | Bold (700) | 1.1 | Hero headlines |
| **H1** | 36px | Bold (700) | 1.2 | Page titles |
| **H2** | 28px | Semibold (600) | 1.3 | Section headers |
| **H3** | 22px | Semibold (600) | 1.3 | Subsections |
| **H4** | 18px | Medium (500) | 1.4 | Card titles |
| **Body** | 16px | Regular (400) | 1.5 | Paragraphs |
| **Body Small** | 14px | Regular (400) | 1.5 | Secondary content |
| **Caption** | 12px | Medium (500) | 1.4 | Labels, timestamps |
| **Micro** | 11px | Medium (500) | 1.3 | Badges, tags |

### 4.3 Number Typography

For financial figures:
- Use **tabular numbers** (font-feature-settings: "tnum")
- Large amounts: Bold weight
- Currency symbol: Same size as numbers
- Decimal places: 2 for Naira, variable for USD

**Example:**
```

N47250000       (bad)
```

## 8. Motion & Animation

### 8.1 Principles

- **Purposeful**: Motion should guide, not distract
- **Quick**: Under 300ms for most transitions
- **Smooth**: Use easing curves, never linear
- **Subtle**: Prefer opacity/transform over complex animations

### 8.2 Standard Durations

| Duration | Use |
|----------|-----|
| 100ms | Micro-interactions (buttons, hovers) |
| 200ms | Small transitions (dropdowns, tooltips) |
| 300ms | Medium transitions (modals, page elements) |
| 500ms | Large transitions (page changes, celebrations) |

### 8.3 Easing Curves

- **Ease-out**: For elements entering (things coming in)
- **Ease-in**: For elements exiting (things going out)
- **Ease-in-out**: For elements moving (transforms within view)

### 8.4 Loading States

- Use skeleton screens, not spinners, for content
- Use spinners only for actions (saving, syncing)
- Pulse animation for skeletons: 2s duration, infinite

## 9. Accessibility

### 9.1 Color Contrast

- Text on backgrounds: Minimum 4.5:1 ratio
- Large text (18px+): Minimum 3:1 ratio
- Interactive elements: Clear focus indicators

### 9.2 Interactive Elements

- Minimum touch target: 44x44 pixels
- Focus visible: 2px purple outline with 2px offset
- Skip links: Available for keyboard navigation

### 9.3 Content

- Alt text for all images
- ARIA labels for icons without text
- Error messages associated with inputs
- Form labels always visible (not placeholder-only)
